什么是PITR

2024-06-11
  • PITR 全称 Point-in-Time Recovery
  • 是在8.0版本中实现的
  • 它利用base backup 和 归档日志，可以恢复数据库到任意一个时间点
  • 在数据库底层存储出现错误的时候，可以使用这个功能进行数据恢复

什么是Base Backup

  • 在线数据库备份分为逻辑备份和物理备份。
  • 逻辑备份有个缺点，备份和恢复时间长
  • 物理备份要比逻辑备份快很多
  • base backup是全量的物理备份
  • base backup过程：
    • 执行pg_backup_start
    • 打snapshot
    • 执行pg_backup_stop
  • base backup 更简单的方法是执行pg_basebackup
