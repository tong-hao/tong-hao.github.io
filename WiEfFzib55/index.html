nebula-模糊查询

2025-07-28 1 min read # nebula # 图数据库

NebulaGraph的模糊查询依赖与ElasticSearch。

ES的存储格式

  • DocID设计：partId + tagId/edgetype + 属性名 + 属性值
  • Doc Fields设计：点ID/边ID

查询过程

  1. 从ES中根据属性值模糊查找，找到一组頂点ID或边ID。
  2. 将頂点ID序列化成对应的Key，根据Key从Nebula的kv存储中完全匹配出对应的Value。
